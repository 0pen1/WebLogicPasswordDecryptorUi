## WeblogicPassword 在线解密(转)

>  原文：https://xz.aliyun.com/t/2149

如有webshell可进行在线解密
把```weblogicdecryptor.jsp```放入服务器进行访问，s="" 中填写 weblogic 的加密密码，可在 boot.properties 文件中找到。默认路径为```../../../Server/security/boot.properties```
